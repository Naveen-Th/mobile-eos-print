## üéØ Design System Values

### Spacing Scale (8px base)
```
4px   = 0.25 rem (minimal)
8px   = 0.5 rem  (tiny)
12px  = 0.75 rem (small)
16px  = 1 rem    (base)
20px  = 1.25 rem (medium)
24px  = 1.5 rem  (large)
32px  = 2 rem    (xlarge)
40px  = 2.5 rem  (2xlarge)
48px  = 3 rem    (3xlarge)
```

### Border Radius Scale
```
12px = Small buttons, inputs, small cards
16px = Main buttons, standard cards, containers
20px = Large cards, main content areas
24px = Icon circles, special elements
```

## ‚ú® Design Principles

1. **8px Grid System**
   - All spacing uses multiples of 4px (preferably 8px)
   - Maintains visual rhythm
   - Easy to scale and adjust

2. **Consistent Hierarchy**
   - Larger gaps between major sections
   - Medium gaps between related elements
   - Small gaps within grouped items

3. **Breathing Room**
   - 24px minimum for screen edges
   - 20-24px for cards and containers
   - Generous padding for touch targets

4. **Touch Targets**
   - Minimum 44px for interactive elements
   - Icons: 48-52px for primary actions
   - Buttons: 18px+ vertical padding

5. **Border Radius**
   - 16px standard for consistency
   - 20px for large cards
   - 12px for small elements
   - 24px for special shapes (icons)
